# Gemini Chatbot

## Technologies Used

- Next.js
- React
- Tailwind CSS

## Getting Started

1.  Clone the repository

```bash
git clone https://github.com/shoaib2000857/TeamGChatBot.git
cd TeamGChatBot
```

2.  Install dependencies:

```bash
npm install
```

3.  Run the development server:

```bash
npm run dev
```

4.  Open your browser and go to

```bash
http://localhost:3000
```
